# 액션-보상-진행 루프 작성 가이드

## 1. 목적 (Purpose)
게임의 핵심 루프(액션-보상-진행)를 일관된 기준으로 설계하여 플레이어의 지속적인 참여와 만족도를 극대화하기 위한 가이드입니다.

## 2. 범위 (Scope)
- **포함**: 코어 루프 설계, 피드백 메커닉, 진행 동기부여, 루프 최적화
- **제외**: 구체적인 보상 수치, UI/UX 구현, 기술적 구현 방법

## 3. 설계 목표 (Design Goals)

1. **명확한 코어 루프 (Clear Core Loop)**: 플레이어가 행동 → 보상 → 진행의 순환을 명확히 이해하고 반복할 수 있도록 설계합니다.
2. **즉각적 피드백 (Immediate Feedback)**: 모든 행동에 대한 즉각적이고 의미 있는 응답을 제공합니다.
3. **지속적 동기부여 (Continuous Motivation)**: 단기·중기·장기 목표의 균형으로 지속적인 참여를 유도합니다.
4. **루프 최적화 (Loop Optimization)**: 반복 플레이에도 지루하지 않은 변화와 깊이를 제공합니다.

## 4. GDD 문서 구조 (Structure)

### 4.1 코어 루프 (Core Loop)
- **기본 루프**: 행동 → 피드백 → 보상 → 진행의 기본 사이클
- **루프 변형**: 상황별·레벨별 루프 변화

### 4.2 피드백 & 보상 (Feedback & Rewards)
- **피드백 메커닉**: 시각적·청각적·게임플레이적 응답
- **보상 구조**: 즉각적·단기·장기 보상 분배

### 4.3 진행 & 최적화 (Progression & Optimization)
- **진행 동기**: 명확한 목표와 마일스톤
- **루프 최적화**: 반복성과 변화의 균형

### 4.4 선택 및 심화 (Optional & Advanced)
- **A/B 테스트 전략**: 루프 타이밍 최적화(5초 vs 10초 vs 15초), 보상 비율 조정(즉각 vs 단기 vs 장기), 피드백 강도 비교
- **리텐션 저하 대응**: 이탈 구간 히트맵 분석, 루프 병목 지점 개선, 보상 주기 재조정, 난이도 곡선 완화
- **장르별 루프 타이밍**: RPG(10-15분 루프), 슈터(30초-2분), 전략(5-10분), 퍼즐(1-3분), 각 장르 최적 타이밍 참조
- **보상 인플레이션 방지**: 파워 커브 상한선 설정, 희소성 유지 전략, 신규 콘텐츠 투입 주기 계획
- **데이터 기반 최적화**: 루프 완료율 추적(목표: 70% 이상), 평균 세션 시간 모니터링, 보상 만족도 설문, 주간 단위 조정

## 5. 관련 개념 (Key Concepts)

### 5.1 루프 설계
- **Core Loop**: 게임의 기본적인 플레이 사이클 (행동 → 피드백 → 보상 → 진행)
- **Feedback Loop**: 시스템 변화가 다시 시스템에 영향을 미치는 순환
- **Flow State**: 도전과 기술의 균형으로 달성하는 몰입 상태

### 5.2 보상 시스템
- **Reward Schedule**: 보상 제공 주기와 패턴 (고정 vs 변동, 즉각 vs 지연)
- **Intrinsic Reward**: 내적 보상 (재미, 성취감, 마스터리)
- **Extrinsic Reward**: 외적 보상 (아이템, 경험치, 화폐)

### 5.3 최적화 & 분석
- **Retention Rate**: 플레이어 유지율 (D1, D7, D30)
- **Session Length**: 평균 세션 길이 및 플레이 빈도
- **Loop Completion**: 루프 완료율 및 이탈 지점 분석

## 6. 품질 체크리스트 (Quality Checklist)

### 6.1 코어 루프
- [ ] 코어 루프가 명확하고 직관적인가?
- [ ] 루프의 각 단계가 플레이어에게 명확히 전달되는가?
- [ ] 루프 반복이 지루하지 않도록 변화가 제공되는가?

### 6.2 피드백 & 보상
- [ ] 모든 행동에 즉각적인 피드백이 제공되는가?
- [ ] 즉각적·단기·장기 보상의 균형이 적절한가?
- [ ] 보상이 플레이어 노력에 비례하는가?

### 6.3 진행 & 최적화
- [ ] 진행 목표가 명확하고 달성 가능한가?
- [ ] 루프가 플레이어의 스킬 향상과 함께 진화하는가?
- [ ] 플레이테스트 데이터로 루프 효과성이 검증되었는가?

## 7. 예시 및 안티패턴 (Examples & Anti-Patterns)

**Case 1: 명확한 코어 루프**
- **좋은 예 (Good)**: "적 처치 → 경험치·아이템 획득 → 레벨업·장비 강화 → 더 강한 적 도전, 각 단계마다 명확한 피드백과 보상"
- **안티패턴 (Bad)**: "행동과 보상의 연결이 불명확하여 플레이어 혼란"

**Case 2: 점진적 보상 (Dark Souls 2)**
- **좋은 예 (Good)**: "즉각적(소울 획득) + 단기(장비 해금) + 장기(지역 개방)의 계층적 보상 구조, 탐험 자체가 보상"
- **안티패턴 (Bad)**: "모든 보상이 즉각적이어서 장기 동기 부족"

**Case 3: 루프 최적화 (Roguelike)**
- **좋은 예 (Good)**: "기본 루프 유지 + 무작위 요소로 매 플레이 변화 제공 + 메타 프로그레션으로 장기 성장"
- **안티패턴 (Bad)**: "완전히 동일한 루프 반복으로 지루함 유발"

**Case 4: 계층적 보상 구조 (Hades)**
- **좋은 예 (Good)**: "매 적 처치마다 즉각적 피드백(경험 수치 표시, 하트 +2) → 웨이브 클리어 시 보상 선택(강화 아이템 3개 중 1개 선택, 5초 이내 의사결정) → 던전 보스 처치 시 장기 보상(영구 언락, 메타 진행도 +3%, 약 3-5분 소요). 루프 타이밍: 적 처치(3초) → 웨이브 클리어(1분) → 보스 전투(5-8분), 루프 완료율 82% 달성"
- **안티패턴 (Bad)**: "모든 보상 단계가 균일하여 특정 이정표에 감정적 반응 없음"

**Case 5: 스로틀된 진행 (Slay the Spire)**
- **좋은 예 (Good)**: "매 턴 카드 선택 결정(즉각적 피드백: 데미지 계산 표시, 방어 수치 변화 가시화) → 매 전투 승리 후 카드 추가 선택(강화도 조정, 3개 중 선택) → 매 3턴마다 릴릭 발견 기회(메타 게임 강화, 비용 250-500골드). 루프 타이밍: 턴당 의사결정(10-15초) → 전투 완료(3-7분) → 보상 선택(30-60초), 의도적 '무릎' 생성으로 긴장-이완 리듬 조성, 평균 세션 길이 45분 유지"
- **안티패턴 (Bad)**: "상수적 피드백으로 선택의 무게감 상실"

**Case 6: 메타 프로그레션 루프 (Dead Cells)**
- **좋은 예 (Good)**: "매 run 진행 중 즉각적 루프(적 처치 → 골드 5-20 획득 → 회복 아이템 구매 결정, 3-5초) → 보스 처치 시 진행 보상(능력 언락 +1, 1회용 아이템 선택, 1분 소요) → 매 run 완료 시 메타 진행(비가 내려오며 영구 강화 포인트 +5-15, 약 2분 소요, 전체 메타 진행도 감정적으로 표시). 5회 run 반복 시 신규 영역 개방(약 4-5시간 플레이 필요). 루프 속도: 빠른 러닝(5초) → 중간 판단(1분) → 느린 메타(2분), 이 3단계의 반복으로 단조로움 방지"
- **안티패턴 (Bad)**: "즉각적 루프와 메타 프로그레션의 연결이 명확하지 않아 플레이어가 진행 의미 이해 실패"

**Case 7: 능력 선택 기반 루프 (Monster Train)**
- **좋은 예 (Good)**: "매 전투 턴마다 카드 배치 선택(즉각적 피드백: 데미지 수치 가시화, 적 HP 바 감소 계산, 2-3초 결정) → 자동 처리 단계에서 결과 수집(주문 발동, 보너스 효과 표시, 5-8초) → 전투 종료 후 정찰꾼(단계별 보상 선택: 새 카드 3개 중 1개, 골드 +50-150, 체력 회복 등, 30-60초) → 매 3전투마다 상점 방문(영구 카드 추가 구매 기회, 1000-3000 금으로 능력 확장). 루프 타이밍: 턴당 판단(3초) × 5턴 → 전투 완료(1분) → 보상 선택(1분) → 메타 강화(30초), 약 30-50회 전투로 1회 run 완료(약 60분). 의도적 '느린 루프' 설계로 각 선택의 전략적 무게 강조"
- **안티패턴 (Bad)**: "모든 회차에서 동일한 보상 구조로 의사결정의 다양성 부족"

**Case 8: 점진적 난이도 루프 (Vampire Survivors)**
- **좋은 예 (Good)**: "매 1-2초마다 적 1-5마리 등장(즉각적 피드백: 화면 밀집도 시각화, HP바 표시, 콤보 카운터) → 플레이어 공격 시 자동 처리로 타격감 제공(파티클, 점수 점프, 0.3초 진동) → 매 30초마다 경험치 구 선택(4가지 옵션 중 1개 선택, 3초 이내 의사결정) → 매 5분마다 주요 보스 출현(스테이지 클리어 보상: 신규 무기 언락 또는 기존 무기 강화). 루프 난이도 곡선: 초반 5분 여유 있는 적 처치 → 중반 10분 밀집도 증가 → 후반 15분+ 극도의 긴장 유지. 60분 플레이 중 마지막 10분이 극적 클라이맥스로 설계하여 성취감 극대화. 메타: 매 run 선택이 다음 run 능력 선택폭 증가(약 50개 능력 점진적 언락)"
- **안티패턴 (Bad)**: "보상과 난이도 증가가 동시에 이루어져 플레이어 적응 불가"

**Case 9: 선택 기반 비용-보상 트레이드오프 (Peglin)**
- **좋은 예 (Good)**: "매 턴마다 핀볼 패그린 발사 선택(즉각적 피드백: 물리 엔진으로 핀 튕김, 적중 수치 표시, 1-2초) → 적 처치 시 아이템 선택(3개 옵션: 공격 +10%, 방어 +5%, 특수능력 추가, 15-30초 고민) → 매 보스 처치 후 약효 선택(강력하지만 부작용 있는 옵션: '공격 +30% but 방어 -10%', 대수롭지 않은 옵션: '최대체력 +2') → 매 층(약 3-5전투) 완료 시 상점 방문(보유 금화로 능력 추가 구매, 의존도 조정). 루프 타이밍: 턴당 발사(1-2초) → 보상 선택(15초) × 3-5회 → 보스전(2-3분) → 층 완료(10초). 약 30분 run에서 20-30번의 의사결정으로 다양한 빌드 창발. 게임오버 시 메타 진행(신규 약효 언락, 약 5-10회 run 반복 시 신규 능력 개방)"
- **안티패턴 (Bad)**: "모든 선택지의 장단점이 불명확하여 전략적 고민이 아닌 운의 영역으로 느껴짐"

## 8. 핵심 인사이트 (Key Insights)

- **코어 루프 명확성 (Rules of the Game)**: 플레이어가 행동·피드백·보상의 순환을 명확히 이해하도록 설계하세요.
- **점진적 보상 구조 (Dark Souls 2)**: 즉각적·단기·장기 보상의 균형을 유지하고, 경험 자체가 보상이 되도록 하세요.
- **플로우 상태 유지 (Csikszentmihalyi)**: 도전과 기술의 균형으로 플레이어 몰입을 극대화하세요.
- **루프 진화 (Roguelike)**: 기본 루프를 유지하면서도 변화와 깊이를 제공하여 반복성을 극복하세요.
