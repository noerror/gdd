# UI 메뉴 & 플로우 작성 가이드

## 1. 목적 (Purpose)
게임의 UI 메뉴와 플로우를 체계적으로 설계하여 플레이어가 원하는 기능에 빠르고 직관적으로 접근할 수 있도록 하기 위한 가이드입니다.

## 2. 범위 (Scope)
- **포함**: 메뉴 구조, 네비게이션 플로우, 화면 전환, 사용자 경로, 백 버튼 처리
- **제외**: 구체적인 그래픽 디자인, 애니메이션 타이밍, 코드 구현

## 3. 설계 목표 (Design Goals)

1. **빠른 접근 (Quick Access)**: 플레이어가 원하는 기능에 최소 클릭으로 도달할 수 있도록 합니다.
2. **명확한 네비게이션 (Clear Navigation)**: 현재 위치와 이동 경로가 항상 명확합니다.
3. **일관된 패턴 (Consistent Patterns)**: 게임 전체에서 일관된 메뉴 패턴을 유지합니다.
4. **에러 방지 (Error Prevention)**: 실수로 인한 중요 행동을 방지하는 확인 절차를 제공합니다.

## 4. GDD 문서 구조 (Structure)

### 4.1 메뉴 구조 (Menu Structure)
- **메인 메뉴**: 게임 시작, 설정, 종료 등
- **인게임 메뉴**: 일시정지, 인벤토리, 맵 등

### 4.2 네비게이션 플로우 (Navigation Flow)
- **화면 전환**: 메뉴 간 이동 경로
- **백 버튼**: 이전 화면으로 돌아가기

### 4.3 사용자 경로 (User Paths)
- **주요 시나리오**: 일반적인 사용 경로
- **예외 처리**: 에러 상황 대응

## 5. 관련 개념 (Key Concepts)

- **Navigation Flow**: 화면 간 이동 경로와 구조
- **Breadcrumb**: 현재 위치를 표시하는 네비게이션 보조 도구
- **Modal Dialog**: 사용자 확인이 필요한 팝업 창
- **Back Button**: 이전 화면으로 돌아가는 기능

## 6. 품질 체크리스트 (Quality Checklist)

### 6.1 메뉴 구조
- [ ] 모든 주요 기능이 3클릭 이내에 접근 가능한가?
- [ ] 메뉴 계층 구조가 논리적인가?
- [ ] 자주 사용하는 기능이 빠르게 접근 가능한가?

### 6.2 네비게이션
- [ ] 현재 위치가 항상 명확하게 표시되는가?
- [ ] 백 버튼이 일관되게 작동하는가?
- [ ] 화면 전환이 자연스러운가?

### 6.3 에러 방지
- [ ] 중요한 행동에 확인 대화상자가 제공되는가?
- [ ] 실수로 인한 데이터 손실을 방지하는가?
- [ ] 모든 경로가 테스트되었는가?

## 7. 예시 및 안티패턴 (Examples & Anti-Patterns)

**Case 1: 빠른 접근**
- **좋은 예 (Good)**: "메인 메뉴 → 설정 → 오디오 (2클릭), 인게임 → 인벤토리 (1클릭), 자주 사용하는 기능을 상위 메뉴에 배치"
- **안티패턴 (Bad)**: "기본 기능에 접근하기 위해 5단계 이상의 메뉴를 거쳐야 함"

**Case 2: 명확한 네비게이션**
- **좋은 예 (Good)**: "현재 메뉴 이름을 상단에 표시, 브레드크럼으로 경로 표시, 백 버튼이 항상 동일한 위치에 배치"
- **안티패턴 (Bad)**: "현재 위치를 알 수 없고, 백 버튼이 화면마다 다른 위치에 배치"

**Case 3: 에러 방지**
- **좋은 예 (Good)**: "게임 종료 시 '저장하지 않은 데이터가 있습니다. 종료하시겠습니까?' 확인, 중요 아이템 판매 시 재확인"
- **안티패턴 (Bad)**: "확인 없이 즉시 실행되어 실수로 인한 데이터 손실"

## 8. 핵심 인사이트 (Key Insights)

- **3클릭 규칙**: 모든 주요 기능이 3클릭 이내에 접근 가능하도록 설계하여 플레이어 편의성을 극대화하세요.
- **일관된 패턴**: 백 버튼, 확인 버튼 등이 항상 동일한 위치와 동작으로 작동하도록 하세요.
- **명확한 피드백**: 화면 전환 시 로딩 상태를 표시하고, 현재 위치를 명확히 전달하세요.
- **에러 방지 우선**: 중요한 행동에는 항상 확인 절차를 제공하여 실수를 방지하세요.
