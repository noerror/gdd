# 게임 피치(Pitch) 문서 작성 가이드라인

이 가이드라인은 **고수준의 컨셉과 비전을 1페이지 안에 효과적으로 전달**하는 게임 피치 문서를 작성하는 방법

## 1. 피치 문서의 핵심 목표 및 대상

- **목표:** 게임의 핵심적인 매력과 잠재력을 **매우 간결하고 명확하게** 전달하여, 청중(투자자, 퍼블리셔, 핵심 팀원 등)의 즉각적인 관심과 흥미를 유발하는 것.
- **핵심:** "이 게임은 무엇이고, 왜 특별하며, 왜 플레이해야 하는가?"에 대한 가장 핵심적인 답변을 제공.

## 2. 피치 문서 작성 핵심 원칙 (1페이지 컨셉 중심)

1. **고수준 개념 집중:** 세부적인 기능 나열보다는 게임의 **핵심 컨셉과 전체적인 비전**에 집중합니다.
2. **1페이지 엄수:** 모든 정보는 단 한 페이지 안에 압축적으로 담겨야 합니다. 간결함이 생명입니다.
3. **강력한 첫인상:** "엘리베이터 피치"를 통해 문서의 첫 부분에서 독자의 시선을 사로잡아야 합니다.
4. **시각적 매력:** 컨셉 아트나 핵심적인 참고 이미지를 통해 게임의 분위기와 비주얼 컨셉을 직관적으로 전달합니다.
5. **핵심 가치 전달:** USP와 디자인 기둥을 통해 게임의 독창성과 지향점을 명확히 합니다.

## 3. 피치 문서 필수 구성 요소 (1페이지 컨셉 중심)

- **구성**: 다음은 1페이지 피치 문서에 반드시 포함되어야 할 핵심 구성 요소입니다. 각 항목은 핵심 키워드나 간결한 문장으로 작성합니다.

### 1. 게임 제목 (Game Title):

- 기억하기 쉽고, 게임의 컨셉을 암시하는 (가제) 제목.

### 2. 게임의 비전 (Game Vision):

- **엘리베이터 피치 (Elevator Pitch)**: 게임의 가장 중요한 핵심 컨셉과 매력을 1~2 문장으로 압축하여 제시합니다. (문서의 가장 첫 부분에 강력하게 배치)
- (예시: "운과 전략이 교차하는 전장, 주사위와 포커의 만남! 포커 족보 기반의 혁신적인 주사위 전투 RPG로, 매 순간 짜릿한 전략적 선택과 극적인 역전의 재미를 선사합니다.")

### 3. 장르 및 핵심 특징 (Genre & Key Features):

- 주요 장르와 게임의 가장 두드러지는 특징을 간결하게 명시.
- (예시: 전략 RPG, 주사위 전투, 로그라이크 덱빌딩 (포커 룰 기반))

### 4. 타겟 고객, 플랫폼 및 마켓 포지션 (Target Audience, Platform & Market Position):

- 주요 타겟 플레이어층과 그들이 이 게임을 좋아할 이유를 간략히 언급.
- 주요 출시 플랫폼 명시.
- **시장 기회** : 시장의 매력과, 시장 내 위치 및 경쟁작과의 차별화된 포지셔닝을 한 줄로 요약.
- (예시: PC(Steam), 전략/RPG 선호 10-30대 / 하드코어 로그라이크와 캐주얼 카드 게임 사이의 미드코어 포지션)
- (예시: 덱빌딩 장르의 성장세와 맞물려, 아직 탐험되지 않은 '포커+RPG' 융합 시장 선점)

### 5. 고유 판매 포인트 (USP - Unique Selling Points):

- **이 게임만이 가진 독창적이고 강력한 매력 2~3가지**를 핵심 키워드나 짧은 문장으로 강조.
- (예시 USP 1: 포커와 RPG의 혁신적 결합 - 익숙함 속 신선함)
- (예시 USP 2: 간결하지만 깊이 있는 전략성 - 쉬운 학습, 깊은 고민)
- (예시 USP 3: 짜릿한 심리전과 역전의 묘미 - 극적인 승리 경험)

### 6. 디자인 기둥 (Design Pillars):

- **프로젝트의 핵심 개념, 지향점, 핵심 경험을 나타내는 2~3가지 키워드 또는 짧은 구절.**
- 모든 디자인 결정의 근간이 되는 원칙.
- (예시 기둥 1: 전략적 선택의 중요성)
- (예시 기둥 2: 운과 실력의 절묘한 조화)
- (예시 기둥 3: 매판 새로운 경험 제공)

### 7. 핵심 시스템 메커니즘 (Core System Mechanics):

- **게임의 가장 핵심적인 시스템이나 메커니즘 1~2가지**를 간결하게 설명. 플레이어가 주로 무엇을 하며 재미를 느끼는지에 대한 설명.
- (예시: 포커 족보 기반 주사위 전투 - 4개 주사위 굴림 (1회 리롤), 족보 완성으로 공격력 결정. 다양한 특수 주사위 수집/강화/장착으로 전략적 덱 빌딩.)

### 8. 플레이어의 최종 목표 (Player's Ultimate Goal):

- 플레이어가 게임을 통해 궁극적으로 달성하고자 하는 목표를 간략하게 제시. 게임의 동기 부여 요소.
- (예시: 최강의 주사위 조합을 완성하여 전설적인 챔피언에 도전하고 승리하는 것. 또는, 모든 스테이지를 클리어하고 숨겨진 엔딩을 발견하는 것.)

### 9. 컨셉 아트 또는 참고 이미지 (Concept Art / Reference Image):

- **게임의 분위기, 비주얼 스타일, 핵심적인 장면을 가장 잘 나타내는 이미지 1~2개**를 포함.
- 자체 컨셉 아트가 없다면, 게임의 느낌을 전달할 수 있는 고품질의 참고 이미지(무드보드)를 활용.
- (예시: `하스스톤`의 보드판, `Dicey Dungeons`의 캐릭터 비주얼)

## 4. 1페이지 피치 문서 작성 팁

- **하나의 핵심 메시지에 집중:** 너무 많은 정보를 담으려 하기보다는, 가장 강력하고 설득력 있는 핵심 메시지를 중심으로 구성합니다.
- **단어 선택에 신중:** 제한된 공간이므로, 모든 단어가 의미를 가져야 합니다. 간결하고 임팩트 있는 단어를 사용합니다.
- **가독성 높은 레이아웃:** 제목, 소제목, 불렛 포인트 등을 활용하여 정보를 명확하게 구분하고 시각적으로 편안하게 읽힐 수 있도록 구성합니다.
- **"그래서 뭘 어쩌라는 거지? (So What?)" 테스트:** 각 항목을 작성한 후, "이 정보가 왜 중요한가?", "이 정보가 청중에게 어떤 의미를 전달하는가?"를 자문자답하며 내용을 다듬습니다.
- **피드백 적극 활용:** 작성 후 여러 사람에게 보여주고 피드백을 받아 객관적으로 문서를 개선합니다. 특히 게임에 대해 전혀 모르는 사람의 피드백이 유용할 수 있습니다.

	# Diablo - 게임 피치 문서

	## 게임의 비전
	**"핵 앤 슬래시의 부활, 매번 새로운 던전!"**
	멀티미디어 화려함이 아닌 순수한 게임플레이로 돌아갑니다.
	마우스 클릭만으로 즐기는 직관적인 액션 RPG.
	무작위 생성 던전으로 플레이할 때마다 완전히 다른 모험을 경험하며, 악마 디아블로를 추적하는 어두운 여정.

	## 장르 및 핵심 특징
	- **장르:** 액션 RPG / 핵 앤 슬래시 / 던전 크롤러
	- **핵심 특징:** 무작위 던전 생성 / 3/4 등각 투시 / 마우스 전용 컨트롤 / 다크 판타지 / 멀티플레이어

	## 타겟 고객 및 플랫폼
	- **타겟:** PC, 액션/RPG 선호 15-35대, 복잡한 인터페이스 없이 빠른 전투와 탐험을 원하는 게이머
	- **플랫폼:** PC (DOS/Windows)
	- **왜 좋아할까:** 간단한 조작 / 깊은 전략 / 끝없는 리플레이 / 즉각적인 만족감

	## 고유 판매 포인트 (USP)

	**USP 1: 동적 무작위 레벨 생성 (DRLG)**
	- 입장할 때마다 새로운 던전 생성 / 방·복도·함정·보물·몬스터 무작위 배치
	- 똑같은 게임은 존재하지 않음 / 무한한 리플레이 가치

	**USP 2: 마우스만으로 모든 것을**
	- 클릭으로 이동 / 클릭으로 공격 / 클릭으로 마법 / 클릭으로 상호작용
	- 키보드 명령 불필요 / 직관적이고 빠른 플레이

	**USP 3: 확장팩 생태계**
	- Magic: The Gathering 모델 차용 / 저가 확장팩 ($4.95)
	- 새로운 아이템·몬스터·함정 지속 추가 / 수집 가치 / 장기 수익 모델

	## 디자인 기둥

	**기둥 1: 순수한 게임플레이**
	- 불필요한 복잡성 제거 / 탐험·전투·캐릭터 성장에 집중 / 멀티미디어보다 재미

	**기둥 2: 즉각적인 만족감**
	- 빠른 전투 / 지속적인 보상 / 명확한 진행감 / "한 판만 더" 중독성

	**기둥 3: 무한한 확장성**
	- DRLG 엔진 기반 콘텐츠 통합 / 확장팩으로 지속 성장 / 오픈 엔딩 구조

	## 핵심 시스템 메커니즘

	**던전 탐험:**
	- 3/4 등각 투시 다이아몬드 그리드 / 16레벨 깊이 / 턴제→실시간 전환
	- 조명·가시거리 시스템 / 적외선 마법 / 세트피스 + 무작위 구역

	**전투 시스템:**
	- 마우스 좌클릭 : 이동·공격·상호작용 / 마우스 우클릭 : 주문 시전
	- 액션 포인트 기반 / 근접·원거리·마법 공격 / 장비 기반 능력 변화

	**캐릭터 성장:**
	- 종족 5-6종 선택 (언덕인·숲인 등) / 직업 3종 (전사·도적·마법사)
	- 힘·마법적성·민첩성·활력 스탯 / 마법 학파 4종 / 레벨업으로 능력 해금

	**마을 허브:**
	- 던전 탐험 사이 귀환 / 잡화점 (장비 구매·판매·수리) / 사원 (치료) / 집 (휴식·보관)

	**2인 / 네트워크 협력:**
	- 모뎀·네트워크 연결 / 시간 제한 턴제 (빠른 결정 강제)
	- 투기장 던전 : PvP 모드 / 친구와 캐릭터 대결	

	## 플레이어의 최종 목표

	**복수와 정화:**
	- 가족을 죽인 악마 디아블로 추적 / 16레벨 던전 끝 디아블로 처치
	- 매 플레이마다 다른 경로·아이템·전투로 새로운 경험
	- **영구 죽음:** 캐릭터 사망 시 완전 삭제, 처음부터 재시작

	## 세계관 & 무드

	**다크 판타지 중세:**
	- 버려진 고딕 교회 → 대리석 무덤 → 지하 묘지 → 자연 동굴 → 악마의 전당 → 디아블로의 방
	- 언데드 주력 (해골·좀비·유령) / 심연의 악마 / 자연 생물 (거미·뱀)
	- 종교/악마 테마 아이템 : 메피스토펠레스 망토 / 성스러운 철퇴

	**Anti-References:**
	- 밝고 화려한 판타지 X / 복잡한 UI X / 느린 턴제 X

	## 비즈니스 모델

	**본편 + 확장팩 전략:**
	- 본편 : 풀 프라이스 ($40-50)
	- 확장팩 : 저가 디스크 ($4.95) / 계산대 POS 아이템
	- 각 팩 : 희귀 아이템 1 + 일반 아이템 8 + 몬스터 2 + 함정 1 + 그래픽 요소
	- 수집 가치 카드 포함 / 시리즈 테마 (Dark Nightmare·Holy Avengers 등)
	- Magic: The Gathering 모델 차용 / 지속 수익·리인게이지먼트

	## 컨셉 아트 / 무드보드

	**참고 이미지:**
	- 고딕 대성당·지하 묘지·악마 전당
	- Ultima·Diablo 이전 등각 RPG
	- 다크 판타지 일러스트 (해골·악마·어둠)

	**비주얼 컨셉:**
	- 640x480 Super VGA / 8방향 애니메이션
	- 어두운 조명·극적인 그림자 / 장비 반영 캐릭터 스프라이트
	- 미니멀 UI / 시네마틱 오프닝

	## 개발 범위 및 팀

	**팀 구성:**
	- 디자이너 1 / 수석 프로그래머 1 / 주니어 프로그래머 2 / 아트 디렉터 2 / 일러스트 1 / 픽셀 아티스트 3 / 사운드 1

	**개발 우선순위:**
	- 직관적 인터페이스 / DRLG 엔진 / 셸 구조 / 사운드 드라이버
	- 4개월 이상 플레이 밸런스·버그 테스트

	**아트 요구사항:**
	- 몬스터 16종 (각 50프레임+) / 캐릭터 다수 프레임 (무기·방어구 반영)
	- 레벨 타입 5-6종 / 마을 / 시네마틱 / UI 스크린
